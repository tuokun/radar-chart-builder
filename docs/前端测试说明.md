# 前端测试说明

**测试目标**：全面测试前端功能，验证前后端联调是否正常，确保用户体验流畅。

**测试环境**：
- 后端：已启动（端口 8081）
- 前端：待启动（端口 5173）
- 浏览器：Chrome/Edge/Safari（推荐 Chrome）

---

## 第一章：环境准备与启动

### 1.1 启动前端开发服务器

**我做了什么**：
执行命令启动前端开发服务器。

**你能看到的现象**：
- 在终端看到启动日志：
  ```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.x.x:5173/
  ```
- 浏览器自动打开（或手动访问 `http://localhost:5173`）

**你要如何进行下去**：
打开浏览器，访问 `http://localhost:5173`

---

### 1.2 访问首页 - 未登录状态

**我做了什么**：
你访问首页，系统检测到未登录。

**你能看到的现象**：
- 页面自动跳转到登录页：`http://localhost:5173/login`
- 看到登录表单界面，包含：
  - 页面标题：登录
  - 用户名输入框
  - 密码输入框
  - 登录按钮
  - 注册链接（文字："还没有账号？立即注册"）

**你要如何进行下去**：
保持当前页面，准备进行登录功能测试。
