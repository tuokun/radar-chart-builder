## 概述

实现雷达图生成器的核心功能，支持用户创建雷达图、配置维度、添加多组数据系列。

## 数据模型

### 实体关系

```
RadarChart (1) ──── (N) Dimension
    │
    └── (N) DataSeries ──── (N) SeriesData ──── (1) Dimension
```

### 实体定义

#### RadarChart（雷达图）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 主键 |
| userId | Long | 所属用户ID |
| title | String | 图表标题 |
| description | String | 图表描述（可选） |
| createTime | LocalDateTime | 创建时间 |
| updateTime | LocalDateTime | 更新时间 |

#### Dimension（维度）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 主键 |
| radarChartId | Long | 所属雷达图ID |
| name | String | 维度名称（如"速度"、"力量"） |
| minValue | Double | 维度最小值（如-10） |
| maxValue | Double | 维度最大值（如100） |
| orderIndex | Integer | 显示顺序 |

#### DataSeries（数据系列）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 主键 |
| radarChartId | Long | 所属雷达图ID |
| name | String | 系列名称（如"2023年"、"小明"） |
| createTime | LocalDateTime | 创建时间 |

#### SeriesData（数据点）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Long | 主键 |
| seriesId | Long | 所属数据系列ID |
| dimensionId | Long | 关联维度ID |
| value | Double | 该维度上的数值 |

## API设计

### 基础路径
`/api/radar-charts`

### 端点列表

| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /api/radar-charts | 创建雷达图（含维度配置） |
| GET | /api/radar-charts | 获取当前用户的雷达图列表 |
| GET | /api/radar-charts/{id} | 获取雷达图详情（含维度、系列、数据） |
| PUT | /api/radar-charts/{id} | 更新雷达图基本信息（标题、描述） |
| DELETE | /api/radar-charts/{id} | 删除雷达图 |
| POST | /api/radar-charts/{id}/series | 添加数据系列 |
| GET | /api/radar-charts/{id}/series | 获取所有数据系列 |
| PUT | /api/radar-charts/{id}/series/{seriesId} | 更新系列名称 |
| DELETE | /api/radar-charts/{id}/series/{seriesId} | 删除数据系列 |
| PUT | /api/radar-charts/{id}/series/{seriesId}/data | 更新系列数据值 |
| GET | /api/radar-charts/{id}/export | 获取图表数据（供前端ECharts渲染） |

### 请求/响应示例

#### 创建雷达图
**请求：** POST /api/radar-charts
```json
{
  "title": "学生能力评估",
  "description": "期末考试成绩分析",
  "dimensions": [
    {"name": "语文", "minValue": 0, "maxValue": 100, "orderIndex": 0},
    {"name": "数学", "minValue": 0, "maxValue": 100, "orderIndex": 1},
    {"name": "英语", "minValue": 0, "maxValue": 100, "orderIndex": 2}
  ]
}
```

**响应：** 201 Created
```json
{
  "id": 1,
  "title": "学生能力评估",
  "description": "期末考试成绩分析",
  "dimensions": [
    {"id": 1, "name": "语文", "minValue": 0, "maxValue": 100, "orderIndex": 0},
    {"id": 2, "name": "数学", "minValue": 0, "maxValue": 100, "orderIndex": 1},
    {"id": 3, "name": "英语", "minValue": 0, "maxValue": 100, "orderIndex": 2}
  ],
  "series": [],
  "createTime": "2026-02-16T10:00:00",
  "updateTime": "2026-02-16T10:00:00"
}
```

#### 添加数据系列
**请求：** POST /api/radar-charts/1/series
```json
{
  "name": "小明",
  "data": [
    {"dimensionId": 1, "value": 85},
    {"dimensionId": 2, "value": 92},
    {"dimensionId": 3, "value": 78}
  ]
}
```

**响应：** 201 Created
```json
{
  "id": 1,
  "name": "小明",
  "data": {
    "1": 85,
    "2": 92,
    "3": 78
  },
  "createTime": "2026-02-16T10:05:00"
}
```

#### 导出图表数据（ECharts）
**请求：** GET /api/radar-charts/1/export

**响应：** 200 OK
```json
{
  "dimensions": ["语文", "数学", "英语"],
  "series": [
    {
      "name": "小明",
      "data": [85, 92, 78]
    },
    {
      "name": "小红",
      "data": [92, 88, 95]
    }
  ]
}
```

## DTO设计

### Param（请求参数）

| 类名 | 用途 |
|------|------|
| CreateRadarChartParam | 创建雷达图 |
| UpdateRadarChartParam | 更新雷达图基本信息 |
| AddDataSeriesParam | 添加数据系列 |
| UpdateSeriesNameParam | 更新系列名称 |
| UpdateSeriesDataParam | 更新系列数据 |

### Result（响应结果）

| 类名 | 用途 |
|------|------|
| RadarChartResult | 雷达图详情 |
| DimensionResult | 维度信息 |
| SeriesResult | 数据系列信息 |
| ChartDataResult | ECharts格式图表数据 |

## 业务规则

1. **权限校验**：所有操作需验证 userId 是否为图表所有者
2. **维度限制**：每个雷达图至少需要3个维度
3. **数据验证**：数据值必须在维度的 minValue 到 maxValue 范围内
4. **级联删除**：
   - 删除雷达图 → 自动删除维度、系列、数据
   - 删除系列 → 自动删除该系列的所有数据点
5. **排序**：维度按 orderIndex 排序返回

## 错误处理

| 错误场景 | 异常类型 | HTTP状态码 |
|---------|---------|-----------|
| 图表不存在 | ResourceNotFoundException | 404 |
| 无权操作 | BadRequestException | 403 |
| 维度数量<3 | BadRequestException | 400 |
| 数据值超出范围 | BadRequestException | 400 |
| 系列名称重复 | BadRequestException | 400 |
| 维度不存在 | BadRequestException | 400 |

## 技术实现

- **持久化**：MyBatis-Plus
- **认证**：JWT Token（从现有认证模块复用）
- **API文档**：Knife4j注解
- **事务管理**：Spring @Transactional
