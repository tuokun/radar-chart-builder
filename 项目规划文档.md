# 项目整体规划文档

## 项目概述
雷达图生成器是一个在线工具，允许用户创建自定义维度的雷达图，并提供模板分享功能。用户可以：
- 自定义雷达图维度和参数
- 可视化生成多边形雷达图
- 分享自己创建的维度模板（待定）
- 使用他人分享的模板快速生成图表（待定）

## 技术栈
**后端：**
- SpringBoot 3.5 + Java 21
- MariaDB 11.4.7
- JWT认证

**前端（本阶段目标）：**
- Web端（主）

**前端（待定）：**
- 微信小程序

## 系统架构
采用前后端分离架构：
- 后端：RESTful API服务，处理数据存储和业务逻辑
- 前端：Web端，通过API与后端交互

## 功能模块
1. **用户模块**（MVP）
   - 注册/登录
   - 个人信息管理

2. **雷达图模块**（核心）
   - 维度自定义
   - 参数配置
   - 图表生成与展示

3. **模板模块**（待定）
   - 模板创建与分享
   - 模板浏览与使用

## 数据库设计（概要）
- 用户表
- 雷达图表
- 维度配置表
- 模板表（待定）
