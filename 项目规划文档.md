# 项目整体规划文档

## 项目概述
雷达图生成器是一个在线工具，允许用户创建自定义维度的雷达图，并支持多数据系列对比展示。用户可以：
- 自定义雷达图维度和参数（名称、最小值、最大值）
- 添加多个数据系列进行对比
- 可视化生成多边形雷达图
- 实时预览和编辑

## 技术栈

- SpringBoot 3.5 + Java 21

- MyBatis-Plus 3.5.16

**前端：**

## 系统架构
采用前后端分离架构：
- **后端：** RESTful API 服务，处理数据存储和业务逻辑
- **前端：** Web 端，通过 API 与后端交互
- **部署：** Docker Compose 编排，前端 Nginx 代理后端 API

## 功能模块

### 1. 用户模块
- 注册/登录
- JWT 认证
- 个人信息管理

### 2. 雷达图模块
- 维度自定义（名称、最小值、最大值、顺序）
- 参数配置
- 图表生成与展示
- 实时预览

### 3. 数据系列模块
- 多数据系列支持
- 系列颜色设置
- 系列数据管理
- 多系列对比展示

### 4. 模板模块（待定）
- 模板创建与分享
- 模板浏览与使用

## 数据库设计

采用四表关联结构：

1. **radar_chart** - 雷达图表
   - id, title, description, create_time, update_time, user_id

2. **dimension** - 维度配置
   - id, chart_id, name, min_value, max_value, order_index

3. **data_series** - 数据系列
   - id, chart_id, name, color, order_index

4. **series_data** - 系列数据
   - id, series_id, dimension_id, value

**特点：**
- 支持多数据系列对比
- 维度支持自定义最小值/最大值（非0起点）
- 使用雪花ID（ASSIGN_ID）
- 自动填充创建/更新时间

## 部署方案

### 本地开发
- 前端：Vite 开发服务器 (localhost:5173)
- 后端：Spring Boot (localhost:8081)
- 数据库：远程 MariaDB (192.168.1.109)

### Docker 部署
- **后端镜像：** radar-chart-builder-server:1.0.0 (173MB)
  - 基础镜像：eclipse-temurin:21-jre
  - 构建：Maven 3.9.12 + JDK 21
  - 时区：Asia/Shanghai

- **前端镜像：** radar-chart-builder-web:1.0.0 (26.3MB)
  - 基础镜像：nginx:alpine
  - 构建：Node 24-alpine
  - 时区：Asia/Shanghai

- **编排：** Docker Compose
  - 后端容器：radar-backend
  - 前端容器：radar-frontend
  - 网络：bridge 网络容器间通信
  - 数据库：外部部署（远程 MariaDB）
