# 产品需求文档

## 用户故事
1. 作为新用户，我可以注册账号并登录系统
2. 作为登录用户，我可以创建自定义维度的雷达图
3. 作为登录用户，我可以查看和管理我创建的雷达图
4. 作为登录用户，我可以分享我创建的维度模板给其他用户（待定）
5. 作为登录用户，我可以浏览他人分享的模板并使用（待定）

## 核心功能（MVP）

### 1. 用户认证
- 注册：用户名/邮箱 + 密码
- 登录：用户名/邮箱 + 密码
- 登出
- JWT token验证

### 2. 雷达图创建
- 添加/删除/编辑维度
- 为每个维度设置数值
- 实时预览雷达图
- 保存雷达图

### 3. 雷达图管理
- 查看我的雷达图列表
- 删除雷达图
- 编辑已有雷达图

## 待定功能
- 模板分享与使用
- 雷达图导出（图片/SVG）
- 图表样式自定义

## 用户流程
**创建雷达图流程：**
登录 → 点击创建雷达图 → 添加维度 → 设置参数 → 生成预览 → 保存

- [ ] 保存/删除雷达图
