# 产品需求文档

## 项目概述
雷达图生成器是一个在线工具，允许用户创建自定义维度的雷达图，并支持多数据系列对比展示。

## 用户故事
1. 作为新用户，我可以注册账号并登录系统
2. 作为登录用户，我可以创建自定义维度的雷达图
3. 作为登录用户，我可以为雷达图添加多个数据系列进行对比
4. 作为登录用户，我可以查看和管理我创建的雷达图
5. 作为登录用户，我可以实时预览雷达图效果

## 核心功能

### 1. 用户认证
- 注册：账号 + 密码
- 登录：账号 + 密码
- 登出
- JWT token 认证

### 2. 雷达图创建
- 添加/删除/编辑维度
- 为每个维度设置名称、最小值、最大值
- 支持自定义维度顺序
- 实时预览雷达图
- 保存雷达图

### 3. 数据系列管理
- 为雷达图添加多个数据系列
- 为每个系列设置名称和颜色
- 为每个维度设置数值
- 支持多系列对比展示
- 实时预览多系列效果

### 4. 雷达图管理
- 查看我的雷达图列表
- 预览雷达图（只读视图）
- 编辑雷达图基本信息和维度
- 编辑数据组
- 删除雷达图（带二次确认）

### 5. 用户界面
- 响应式布局（支持桌面端和移动端）
- 直观的维度编辑器
- 直观的数据组编辑器
- 实时预览功能
- 一键预览和编辑模式切换

## 待定功能
- 模板分享与使用
- 雷达图导出（图片/SVG）
- 图表样式自定义
- 微信小程序版本

## 用户流程
**创建雷达图流程：**
```
登录 → 点击创建雷达图 → 设置标题和描述
→ 添加维度（名称、最小值、最大值）
→ 添加数据组（系列名称、颜色）
→ 为各维度设置数值
→ 实时预览 → 保存
```

**编辑雷达图流程：**
```
列表页点击卡片 → 预览页面（只读）
→ 点击"编辑数据" → 编辑模式
→ 修改维度或数据组 → 实时预览 → 保存
```
